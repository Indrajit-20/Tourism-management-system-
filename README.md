Tourism Management System (TSM) ✈️
A full-stack web application designed to streamline the booking and management of domestic travel packages and daily bus ticket booking. Built using the MERN stack for a seamless, responsive user experience. 

🚀 Key Features
Domestic Package Management: Browse and book curated travel packages within the country.

Day-Trip Planning: Specialized modules for short-duration, single-day tours.

User Authentication: Secure login and registration using JWT.

Dynamic Dashboard: Manage bookings and view package details in real-time.

Responsive Design: Optimized for both desktop and mobile users.

🛠️ Tech Stack
Frontend: React.js, Tailwind CSS / Bootstrap

Backend: Node.js, Express.js

Database: MongoDB (Mongoose)

State Management: Context API / Redux

Authentication: JSON Web Tokens (JWT) & Bcrypt


⚙️ Installation & Setup
Clone the project:

Bash
git clone https://github.com/your-username/tourism-management-system.git
cd tourism-management-system
Setup Backend:

Bash
cd backend
npm install
# Create a .env file and add:
# MONGO_URI=your_mongodb_uri
# JWT_SECRET=your_secret_key
npm start
Setup Frontend:

Bash
cd ../frontend
npm install
npm start
